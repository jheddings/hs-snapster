# CLAUDE.md — hs-snapster

## Overview

Snapster is a Hammerspoon Spoon for window management on macOS, providing
hotkey-driven positioning, resizing, and layout management.

## Commit Conventions

Use [Conventional Commits](https://www.conventionalcommits.org/) format:

```
<type>(<scope>): <description>
```

Types: `feat`, `fix`, `chore`, `docs`, `refactor`, `test`, `style`, `perf`

Scope is optional but encouraged (e.g. `fix(layout): ...`, `feat(snap): ...`).

Include the issue number when applicable (e.g. `feat: add grid layout (#4)`).

## Branch Naming

Use the same type prefixes as commits, followed by a short description:

```
<type>/<short-description>
```

Examples: `feat/grid-layout`, `fix/multi-monitor`, `chore/update-deps`
